# Prerequisites

- Node.js is required to install dependencies
- [JSON Schema Ref Parser](https://github.com/APIDevTools/json-schema-ref-parser)

Install using npm:

```
npm install @apidevtools/json-schema-ref-parser
```

# Running the script

By default, file paths assume you are running the script from the base of the repository:

```
node scripts/compile-schemas/compile-schemas.js
```

This will place the files in the `schema/out/` directory.
